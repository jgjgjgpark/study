AWS CDK

Prerequisite

  • Install AWS CLI :IAM User를 생성한 후 권한을 설정하고 CLI를 통해 machine에서 사용할 수 있도록 keyid, secret을 설정
  • Install Node
    사용되는 언어와 상관없이 Node가 사용됨
  • Install AWS CDK
npm install aws-cdk
  • Frist AWS CDK app
mkdir hello-cdk
cd hello-cdk
cdk init app --language typescript
  • Code Program
  • Deploy
cdk ls
cdk synth
cdk deploy
  • Modify Program
cdk diff
cdk deploy
  • Destroy app’s resources
cdk destroy
